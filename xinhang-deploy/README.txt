新航学校网站部署包
==================

此文件夹包含网站部署所需的全部文件。
请按照 QUICK_START.md 中的步骤操作。

文件夹说明：
- QUICK_START.md    现场操作手册（按步骤执行）
- app/              应用程序文件（后端+前端）
- data/             数据文件（新闻+图片）
- system/           系统配置文件
- offline-packages/ 离线安装包（无网络时使用）
- scripts/          自动化安装脚本

注意事项：
1. 所有脚本需要用 sudo 执行
2. 按照编号顺序执行脚本（00 → 01 → 02 → ...）
3. 遇到问题请联系远程技术支持
